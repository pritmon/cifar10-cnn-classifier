# 🖼️ CIFAR-10 CNN Image Classifier

A live demo of a Convolutional Neural Network that sorts 32×32 images into 10
classes: airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck.

Drag in a photo (or click an example) to see the model's top 3 guesses with
confidence scores.

> Trained on the tiny 32×32 CIFAR-10 images, so simple, centred subjects work
> best. Source code and full write-up:
> **https://github.com/pritmon/cifar10-cnn-classifier**
